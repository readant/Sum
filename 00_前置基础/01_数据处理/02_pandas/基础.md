# Pandas 数据处理

## 基本操作

```python
import pandas as pd

# 创建DataFrame
data = {
    "姓名": ["张三", "李四", "王五"],
    "数学": [85, 92, 78],
    "英语": [90, 88, 95]
}
df = pd.DataFrame(data)

# 读取数据
df = pd.read_csv("数据.csv")
df = pd.read_excel("数据.xlsx")
```

## 常用查看

```python
df.head()          # 前5行
df.tail()          # 后5行
df.shape           # 形状 (行, 列)
df.info()          # 信息摘要
df.describe()      # 统计描述
df.columns         # 列名
df.dtypes          # 数据类型
```

## 索引与选择

```python
# 按列选择
df["姓名"]              # 单列
df[["姓名", "数学"]]     # 多列

# 按行选择
df.loc[0]              # 按标签
df.iloc[0:3]           # 按位置

# 条件筛选
df[df["数学"] > 80]    # 数学成绩大于80
df[(df["数学"] > 80) & (df["英语"] > 85)]  # 多个条件
```

## 数据清洗

```python
# 缺失值
df.isnull()            # 检查缺失
df.dropna()            # 删除缺失行
df.fillna(0)           # 填充缺失值

# 去重
df.drop_duplicates()

# 重命名列
df.rename(columns={"旧名": "新名"})
```

## 数据统计

```python
df["数学"].mean()      # 平均值
df["数学"].sum()       # 总和
df["数学"].max()       # 最大值
df["数学"].min()       # 最小值
df["数学"].std()       # 标准差

# 分组统计
df.groupby("班级")["数学"].mean()
```

## 数据保存

```python
df.to_csv("输出.csv", index=False)
df.to_excel("输出.xlsx", index=False)
```
