## 测试模块

### 1. 手部检测测试 (hand_detection)
- **功能**：检测和可视化手部关键点
- **技术**：MediaPipe HandLandmarker
- **特点**：支持双手检测，实时绘制骨骼结构

### 2. 手势识别测试 (gesture_recognition)
- **功能**：识别简单的静态手势
- **技术**：基于规则的手势识别
- **特点**：支持数字0-6和日常用语手势

### 3. 双手检测测试 (dual_hand)
- **功能**：同时检测和区分左右手
- **技术**：左右手自动判断
- **特点**：颜色区分（右手绿色，左手蓝色）

### 4. 置信度估计测试 (confidence_estimation)
- **功能**：评估手势识别的置信度
- **技术**：基于手指状态匹配度计算
- **特点**：颜色编码显示不同置信度水平

## 依赖要求

所有测试程序需要以下依赖：
- Python 3.7+
- opencv-python
- mediapipe
- numpy

## 运行方法

1. 进入对应测试模块目录
2. 运行测试程序：
   ```
   python [test_file].py
   ```
3. 按照每个模块的README.md中的说明进行测试
4. 按ESC键退出测试

## 模型文件

每个测试模块会自动下载所需的模型文件：
- `hand_landmarker.task` - 手部检测模型
- 首次运行时会自动下载到对应测试目录

## 技术特点

- **模块化设计**：每个功能独立测试
- **实时响应**：处理速度快，适合实时应用
- **用户友好**：直观的界面和丰富的视觉反馈
- **无模型依赖**：基于规则识别，不需要训练

## 测试建议

1. 从手部检测开始，确保基础功能正常
2. 然后测试手势识别，熟悉支持的手势
3. 测试双手检测，验证左右手区分
4. 最后测试置信度估计，了解识别可靠性

## 故障排除

- **模型下载失败**：检查网络连接，手动下载模型文件
- **摄像头无法打开**：确保摄像头权限正确，没有被其他程序占用
- **识别效果不佳**：确保光线充足，手部在摄像头视野内
