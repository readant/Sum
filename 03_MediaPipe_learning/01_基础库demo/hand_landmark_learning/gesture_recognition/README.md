# 手势识别测试

## 功能说明

此测试程序用于识别简单的静态手势，是手语识别系统的核心功能模块。

## 技术要点

- 基于手部关键点的规则识别
- 支持数字手势（0-6）
- 支持日常用语手势（Hello, Thank you, OK）
- 使用角度计算判断手指伸直状态
- 使用距离计算区分相似手势

## 文件结构

- `gesture_recognition_test.py` - 主要测试程序
- `hand_landmarker.task` - 手部检测模型（首次运行时自动下载）

## 运行说明

1. 确保已安装必要的依赖：
   - opencv-python
   - mediapipe
   - numpy

2. 运行测试程序：
   ```
   python gesture_recognition_test.py
   ```

3. 测试方法：
   - 将手放在摄像头前
   - 做出不同的手势
   - 观察识别结果
   - 按ESC键退出

## 支持的手势

### 数字手势
- 0: 握拳
- 1: 只有食指伸直
- 2: 食指和中指伸直
- 3: 食指、中指和无名指伸直
- 4: 食指、中指、无名指和小指伸直
- 5: 所有手指伸直
- 6: 拇指和小指伸直

### 日常用语手势
- Hello: 拇指和食指伸直并张开
- Thank you: 只有拇指伸直
- OK: 拇指和食指形成圆圈，其他手指伸直

## 预期效果

- 摄像头画面中会显示手部的骨骼结构
- 顶部显示测试名称
- 左侧显示识别到的手势
- 实时更新识别结果

## 依赖文件

- 模型文件：`hand_landmarker.task`（自动下载）
- 输入：摄像头实时视频流
- 输出：带有关键点标记和识别结果的视频画面
