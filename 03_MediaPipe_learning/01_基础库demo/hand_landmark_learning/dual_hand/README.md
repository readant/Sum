# 双手检测测试

## 功能说明

此测试程序用于同时检测和区分左右手，是实现复杂手语识别的基础。

## 技术要点

- 支持同时检测两只手
- 自动判断左手和右手
- 左右手使用不同颜色区分
- 实时显示手型信息

## 文件结构

- `dual_hand_test.py` - 主要测试程序
- `hand_landmarker.task` - 手部检测模型（首次运行时自动下载）

## 运行说明

1. 确保已安装必要的依赖：
   - opencv-python
   - mediapipe
   - numpy

2. 运行测试程序：
   ```
   python dual_hand_test.py
   ```

3. 测试方法：
   - 将双手放在摄像头前
   - 观察左右手的颜色区分
   - 测试不同手型的识别
   - 按ESC键退出

## 颜色说明

- 右手：绿色
- 左手：蓝色

## 预期效果

- 摄像头画面中会显示两只手的骨骼结构
- 每只手会显示对应的手型标签
- 顶部显示测试名称
- 左侧显示检测到的手的数量
- 底部显示颜色说明

## 依赖文件

- 模型文件：`hand_landmarker.task`（自动下载）
- 输入：摄像头实时视频流
- 输出：带有颜色区分的双手标记画面
