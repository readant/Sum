# 置信度估计测试

## 功能说明

此测试程序用于评估手势识别的置信度，帮助了解识别结果的可靠性。

## 技术要点

- 基于手指状态匹配度计算置信度
- 针对特定手势的额外判断逻辑
- 颜色编码显示不同置信度水平
- 实时更新置信度值

## 文件结构

- `confidence_estimation_test.py` - 主要测试程序
- `hand_landmarker.task` - 手部检测模型（首次运行时自动下载）

## 运行说明

1. 确保已安装必要的依赖：
   - opencv-python
   - mediapipe
   - numpy

2. 运行测试程序：
   ```
   python confidence_estimation_test.py
   ```

3. 测试方法：
   - 将手放在摄像头前
   - 做出不同的手势
   - 观察置信度的变化
   - 测试模糊手势的置信度
   - 按ESC键退出

## 置信度颜色说明

- 绿色：>80%（高置信度）
- 黄色：60%-80%（中等置信度）
- 红色：<60%（低置信度）

## 预期效果

- 摄像头画面中会显示手部的骨骼结构
- 顶部显示测试名称
- 左侧显示识别到的手势和置信度
- 底部显示置信度颜色说明
- 实时更新置信度值

## 依赖文件

- 模型文件：`hand_landmarker.task`（自动下载）
- 输入：摄像头实时视频流
- 输出：带有关键点标记和置信度显示的视频画面
