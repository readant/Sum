# 手部检测测试

## 功能说明

此测试程序用于检测和可视化手部关键点，是手语识别系统的基础模块。

## 技术要点

- 使用MediaPipe HandLandmarker进行手部关键点检测
- 支持同时检测两只手
- 实时绘制手部骨骼结构
- 显示检测到的手的数量

## 文件结构

- `hand_detection_test.py` - 主要测试程序
- `hand_landmarker.task` - 手部检测模型（首次运行时自动下载）

## 运行说明

1. 确保已安装必要的依赖：
   - opencv-python
   - mediapipe
   - numpy

2. 运行测试程序：
   ```
   python hand_detection_test.py
   ```

3. 测试方法：
   - 将手放在摄像头前
   - 观察手部关键点的实时绘制
   - 按ESC键退出

## 预期效果

- 摄像头画面中会显示手部的骨骼结构
- 顶部显示测试名称
- 左侧显示检测到的手的数量
- 双手同时出现时会分别标记

## 依赖文件

- 模型文件：`hand_landmarker.task`（自动下载）
- 输入：摄像头实时视频流
- 输出：带有关键点标记的视频画面
