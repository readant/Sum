# 面部检测测试

## 功能描述

面部检测测试模块用于实时检测和可视化人脸关键点，展示 MediaPipe FaceLandmarker 的功能。

## 技术特点

- **实时面部检测**：使用 MediaPipe FaceLandmarker 实时检测面部关键点
- **可视化展示**：绘制面部骨骼结构，包括眼睛、鼻子、嘴巴等关键点
- **用户友好界面**：清晰的视觉反馈和操作提示

## 测试步骤

1. 确保 `models` 目录中存在 `face_landmarker.task` 模型文件
2. 运行测试程序：
   ```
   python face_detection_test.py
   ```
3. 面对摄像头，确保面部在摄像头视野内
4. 观察屏幕上的面部关键点检测效果
5. 按 ESC 键退出测试

## 预期输出

- 实时显示摄像头画面
- 检测到面部时，在面部绘制关键点和连接线
- 显示检测到的面部数量
- 未检测到面部时，显示相应提示

## 技术参数

- **模型**：MediaPipe FaceLandmarker
- **输入**：摄像头实时视频流
- **输出**：带有面部关键点的视频流
- **检测速度**：实时（取决于硬件性能）

## 故障排除

- **模型文件缺失**：确保 `models` 目录中存在 `face_landmarker.task` 文件
- **摄像头无法打开**：确保摄像头权限正确，没有被其他程序占用
- **检测效果不佳**：确保光线充足，面部朝向摄像头，距离适当
