# 面部特征分析测试

## 功能描述

面部特征分析测试模块用于分析面部特征和属性，包括面部比例、对称性、特征点距离等。

## 技术特点

- **面部比例分析**：分析面部三庭五眼等比例
- **对称性分析**：分析面部的左右对称性
- **特征点距离**：计算面部关键特征点之间的距离
- **可视化展示**：实时显示分析结果

## 测试步骤

1. 确保 `models` 目录中存在 `face_landmarker.task` 模型文件
2. 运行测试程序：
   ```
   python facial_features_test.py
   ```
3. 面对摄像头，确保面部在摄像头视野内
4. 观察屏幕上的面部特征分析结果
5. 尝试不同的角度和表情
6. 按 ESC 键退出测试

## 预期输出

- 实时显示摄像头画面
- 检测到面部时，显示面部特征分析结果
- 显示面部对称性比例
- 显示面部比例分析（三庭五眼）
- 显示特征点之间的距离信息
- 未检测到面部时，显示相应提示

## 技术参数

- **模型**：MediaPipe FaceLandmarker
- **分析类型**：比例分析、对称性分析、距离计算
- **输入**：摄像头实时视频流
- **输出**：带有分析结果标注的视频流
- **分析速度**：实时（取决于硬件性能）

## 故障排除

- **模型文件缺失**：确保 `models` 目录中存在 `face_landmarker.task` 文件
- **摄像头无法打开**：确保摄像头权限正确，没有被其他程序占用
- **分析效果不佳**：确保光线充足，面部正对摄像头
