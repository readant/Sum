# 表情识别测试

## 功能描述

表情识别测试模块用于识别面部表情和情绪状态，展示基于面部关键点的表情分析功能。

## 技术特点

- **多表情识别**：支持多种基本表情的识别
- **情绪分析**：基于面部关键点位置变化分析情绪
- **实时可视化**：实时显示表情类型和情绪状态

## 测试步骤

1. 确保 `models` 目录中存在 `face_landmarker.task` 模型文件
2. 运行测试程序：
   ```
   python expression_recognition_test.py
   ```
3. 面对摄像头，确保面部在摄像头视野内
4. 观察屏幕上的表情识别效果
5. 尝试不同的表情（微笑、皱眉、惊讶等）
6. 按 ESC 键退出测试

## 预期输出

- 实时显示摄像头画面
- 检测到面部时，识别并显示表情类型
- 显示识别的置信度
- 显示情绪状态（开心、悲伤、惊讶等）
- 未检测到面部时，显示相应提示

## 技术参数

- **模型**：MediaPipe FaceLandmarker + 基于规则的表情识别
- **表情类型**：微笑、皱眉、惊讶、自然等
- **输入**：摄像头实时视频流
- **输出**：带有表情标签的视频流
- **检测速度**：实时（取决于硬件性能）

## 故障排除

- **模型文件缺失**：确保 `models` 目录中存在 `face_landmarker.task` 文件
- **摄像头无法打开**：确保摄像头权限正确，没有被其他程序占用
- **识别效果不佳**：确保光线充足，面部表情明显
