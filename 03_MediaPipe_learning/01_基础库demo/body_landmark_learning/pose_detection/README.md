# 人体姿态检测测试

## 功能描述

人体姿态检测测试模块用于实时检测和可视化人体姿态关键点，展示 MediaPipe PoseLandmarker 的功能。

## 技术特点

- **实时姿态检测**：使用 MediaPipe PoseLandmarker 实时检测人体姿态关键点
- **可视化展示**：绘制人体骨骼结构，包括头部、躯干、四肢等关键点
- **用户友好界面**：清晰的视觉反馈和操作提示

## 测试步骤

1. 确保 `models` 目录中存在 `pose_landmarker.task` 模型文件
2. 运行测试程序：
   ```
   python pose_detection_test.py
   ```
3. 站在摄像头前，确保全身或上半身在摄像头视野内
4. 观察屏幕上的姿态关键点检测效果
5. 尝试不同的姿势，观察检测结果
6. 按 ESC 键退出测试

## 预期输出

- 实时显示摄像头画面
- 检测到人体时，绘制姿态关键点和连接线
- 显示检测到的人体数量
- 未检测到人体时，显示相应提示

## 技术参数

- **模型**：MediaPipe PoseLandmarker
- **输入**：摄像头实时视频流
- **输出**：带有姿态关键点的视频流
- **检测速度**：实时（取决于硬件性能）

## 故障排除

- **模型文件缺失**：确保 `models` 目录中存在 `pose_landmarker.task` 文件
- **摄像头无法打开**：确保摄像头权限正确，没有被其他程序占用
- **检测效果不佳**：确保光线充足，人体在摄像头视野内，距离适当
