# 线性代数基础

## 矩阵与向量

```python
import numpy as np

# 向量
v = np.array([1, 2, 3])

# 矩阵
A = np.array([[1, 2], [3, 4]])

# 单位矩阵
I = np.eye(3)

# 零矩阵
Z = np.zeros((3, 3))

# 对角矩阵
D = np.diag([1, 2, 3])
```

## 矩阵运算

```python
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

A + B              # 加法
A - B              # 减法
A * B              # 元素乘法（不是矩阵乘法）
A @ B              # 矩阵乘法
A.T                # 转置
np.linalg.inv(A)   # 逆矩阵
```

## 行列式与秩

```python
np.linalg.det(A)      # 行列式
np.linalg.matrix_rank(A)  # 矩阵秩
```

## 范数（向量的长度）

```python
v = np.array([3, 4])

np.linalg.norm(v)           # L2范数（欧几里得距离）
np.linalg.norm(v, ord=1)     # L1范数
np.linalg.norm(v, ord=np.inf)  # 无穷范数
```

## 特征值与特征向量

```python
 eigenvalues, eigenvectors = np.linalg.eig(A)
```

## 重要概念

| 概念 | 含义 |
|------|------|
| 向量 | 有方向和大小的量 |
| 矩阵 | 二维数组 |
| 行列式 | 矩阵的"面积"或"体积"缩放因子 |
| 逆矩阵 | 满足 A·A⁻¹ = I |
| 特征值 | 矩阵在变换中的缩放因子 |
