# 概率统计基础

## 基础统计量

```python
import numpy as np
import statistics

data = [85, 90, 78, 92, 88, 76, 95, 82]

np.mean(data)           # 平均值：85.75
np.median(data)         # 中位数：86.5
statistics.mode(data)   # 众数
np.var(data)            # 方差
np.std(data)            # 标准差
```

## 概率分布

```python
# 正态分布
from scipy import stats

# 概率密度
x = np.linspace(-3, 3, 100)
y = stats.norm.pdf(x, 0, 1)  # 标准正态分布

# 累积分布
stats.norm.cdf(1.96)   # P(X <= 1.96)

# 分位数
stats.norm.ppf(0.975)  # 97.5%分位数
```

## 常用分布

| 分布 | 用途 |
|------|------|
| 正态分布 | 自然现象、测量误差 |
| 二项分布 | 成功/失败实验 |
| 泊松分布 | 稀有事件发生次数 |
| 均匀分布 | 等概率事件 |

## 假设检验

```python
from scipy import stats

# t检验：检验两组数据均值是否显著不同
group1 = [85, 90, 88, 92]
group2 = [78, 82, 80, 85]
t_stat, p_value = stats.ttest_ind(group1, group2)

# 判断：p < 0.05 则拒绝原假设
```

## 相关性

```python
import pandas as pd

x = [1, 2, 3, 4, 5]
y = [2, 4, 5, 4, 5]

np.corrcoef(x, y)           # 相关系数矩阵
stats.pearsonr(x, y)        # 皮尔逊相关系数
```

## 大数定律与中心极限定理

- **大数定律**：样本量足够大时，样本均值趋近于总体均值
- **中心极限定理**：大量独立随机变量的和趋近于正态分布
