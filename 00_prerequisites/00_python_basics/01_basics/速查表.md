# Python 基础速查

## 变量与数据类型

```python
# 基础类型
a = 10          # int
b = 3.14        # float
c = "hello"     # str
d = True        # bool
e = None        # 空值

# 列表（可变）
nums = [1, 2, 3, 4, 5]
nums.append(6)   # 添加
nums[0]         # 索引

# 元组（不可变）
point = (10, 20)

# 字典
person = {"name": "张三", "age": 20}
person["name"]
person.keys()

# 集合（去重、无序）
s = {1, 2, 3, 3}  # {1, 2, 3}
```

## 字符串操作

```python
s = "Hello World"

s.upper()           # HELLO WORLD
s.lower()           # hello world
s.split()           # ['Hello', 'World']
s.replace("World", "Python")
f"我的名字是{name}"  # 格式化
```

## 类型转换

```python
int("123")      # 字符串转整数
float("3.14")   # 转浮点数
str(100)        # 转字符串
list((1,2,3))   # 转列表
```

## 常用函数

```python
len([1,2,3])        # 长度
range(10)           # 0-9
range(1, 10)        # 1-9
range(0, 10, 2)     # 0,2,4,6,8
sum([1,2,3])        # 求和
max([1,2,3])        # 最大值
min([1,2,3])        # 最小值
sorted([3,1,2])     # 排序
```
