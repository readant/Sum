# 函数

## 基本定义

```python
def say_hello(name):
    return f"你好, {name}"

result = say_hello("张三")
print(result)  # 你好, 张三
```

## 默认参数

```python
def greet(name, greeting="你好"):
    return f"{greeting}, {name}"

greet("李四")              # 你好, 李四
greet("王五", "早上好")    # 早上好, 王五
```

## 关键字参数

```python
def power(base, exponent=2):
    return base ** exponent

power(3)           # 9
power(3, 3)        # 27
power(exponent=3, base=2)  # 8
```

## 变量参数

```python
# *args 接收不定数量位置参数
def sum_all(*nums):
    return sum(nums)

sum_all(1, 2, 3)   # 6
sum_all(1, 2, 3, 4, 5)  # 15

# **kwargs 接收不定数量关键字参数
def print_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

print_info(name="张三", age=20, city="北京")
```

## 匿名函数

```python
# lambda 表达式
square = lambda x: x ** 2
square(5)  # 25

# 配合内置函数
numbers = [3, 1, 4, 1, 5, 9, 2, 6]
sorted(numbers, reverse=True)  # 降序排列
```

## 函数作为参数

```python
def apply_twice(func, x):
    return func(func(x))

def double(x):
    return x * 2

apply_twice(double, 5)  # 20
```
