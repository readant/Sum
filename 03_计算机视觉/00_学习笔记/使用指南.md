# YOLO 使用指南

## 环境激活
```bash
conda activate cv-yolo
```

## 常用命令

### 推理
```python
from ultralytics import YOLO
model = YOLO('yolov8n.pt')
results = model('图片路径')
```

### 摄像头实时检测
```python
model = YOLO('yolov8n.pt')
model.predict(source=0, show=True)
```

### 训练自己的数据
```python
model = YOLO('yolov8n.pt')
model.train(data='数据集.yaml', epochs=100)
```
